## 基本規則

### D&D 5th Edition 規則

本遊戲基於官方 D&D 5th Edition 規則集實現。

#### 擲骰
- **d20**：主要判定用骰子
- **d12, d10, d8, d6, d4**：傷害骰子
- **多次擲骰**：例如 3d6 表示擲 3 個 6 面骰子

#### 成功判定
```
判定結果 = 擲骰 + 修正值 ≥ DC（難度等級）
DC 10 = 簡單
DC 15 = 困難
DC 20 = 極難
```

#### 回合制
每個回合 6 秒的遊戲時間。

### 步驟 3：分配屬性

#### 屬性說明

**力量 (STR)**
- 衡量：體力和肌肉
- 用途：近戰攻擊、運動檢定
- 修正值影響：近戰傷害

**敏捷 (DEX)**
- 衡量：協調和反應
- 用途：護甲等級、閃躲、潛行
- 修正值影響：遠程攻擊、先攻

**體質 (CON)**
- 衡量：健康和耐力
- 用途：生命值、死亡豁免
- 修正值影響：每級額外 HP

**智力 (INT)**
- 衡量：推理和記憶
- 用途：秘聞、調查、自然知識
- 修正值影響：秘聞技能檢定

**感知 (WIS)**
- 衡量：察覺和洞察
- 用途：察覺、醫術、生存
- 修正值影響：感知檢定

**魅力 (CHA)**
- 衡量：個性和影響力
- 用途：勸導、欺騙、威嚇
- 修正值影響：說服和表演

#### 屬性分配方法

**方法 1：擲骰（推薦）**
- 擲 3d6 六次
- 結果應在 3-18 之間
- 將結果分配到各屬性

**方法 2：標準陣列**
```
15, 14, 13, 12, 10, 8
```

**方法 3：自定義**
- 每個屬性 8-15
- 根據種族加成調整

#### 修正值計算

```
修正值 = (屬性值 - 10) / 2（向下取整）

例如：
力量 17 → (17-10)/2 = +3
敏捷 12 → (12-10)/2 = +1
體質 9 → (9-10)/2 = -1
```

### 步驟 4：計算派生屬性

#### 生命值 (HP)
```
最大 HP = 職業基礎 HP + 體質修正值 + (等級-1) × (基礎HP/2 + 體質修正值)
```

**例如**：戰士，體質 16（修正 +3）
```
HP = 10 + 3 + (1-1) × (5 + 3) = 13 HP
```

#### 護甲等級 (AC)
```
基礎 AC = 10 + 敏捷修正值

例如：敏捷 14（修正 +2）
AC = 10 + 2 = 12
```

#### 先攻 (Initiative)
```
先攻 = 敏捷修正值

例如：敏捷 15（修正 +2）
先攻 = +2
```

## 屬性系統

### 屬性檢定

```
檢定結果 = d20 + 屬性修正值

DC 10 = 簡單 (50% 成功概率)
DC 15 = 中等 (25% 成功概率)
DC 20 = 困難 (5% 成功概率)
```

## 戰鬥系統

### 戰鬥回合

1. **確定先攻**
   - 擲 d20 + 先攻修正值
   - 高者先攻擊

2. **行動順序**
   - 按先攻從高到低排列
   - 每回合一次動作

3. **攻擊步驟**
   - 擲 d20 + 攻擊加值
   - 對比敵人 AC
   - 命中則擲傷害骰子

### 攻擊投骰

```
攻擊檢定 = d20 + 屬性修正值 + 精通加成（如適用）

結果 ≥ 敵人 AC = 命中
```

### 傷害投骰

```
傷害 = 傷害骰子 + 屬性修正值

例如：劍 (d8+3)
      擲 d8，添加 +3 力量加成
```

## 進階規則

### 重傷（0 生命值）
- 立即掉落
- 無意識且無法行動
- 開始進行死亡豁免

### 死亡豁免
- 擲 d20
- 10+ = 成功
- 3 次失敗 = 死亡

### 短休（1 小時）
- 恢復一些生命值
- 恢復一些法術位

### 長休（8 小時）
- 恢復所有生命值
- 恢復所有法術位
- 恢復所有資源

## 遊戲大師提示

### 保持故事流動
- 不要被規則卡住
- 規則為故事服務，而不是反之
- 在不確定時，進行 DC 15 檢定

### 平衡難度
- 簡單戰鬥：1 個弱敵人
- 中等：2-3 個標準敵人
- 困難：4+ 個敵人或 1 個強敵人

### 獎勵創意
- 玩家想出創意解決方案？
- 給予優勢擲骰
- 降低 DC
- 允許額外效果

## 常見問題

**Q: 什麼時候需要進行檢定？**
A: 當行動結果不確定且失敗有實際後果時。

**Q: 我可以重新擲骰嗎？**
A: 一般不行，除非有特殊能力或遊戲大師允許。

**Q: 如何計算多個加成？**
A: 所有加成相加，除非特別說明只取一次。

**Q: 我可以自動成功嗎？**
A: 只有在沒有對手的情況下。與對手對抗需要擲骰。
