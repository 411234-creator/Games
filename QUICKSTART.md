# 快速開始指南

## 📋 系統需求

- 現代網頁瀏覽器（Chrome、Firefox、Safari、Edge）
- JavaScript 已啟用
- 互聯網連接（可選，用於 AI API）

## 🚀 5分鐘快速啟動

### 步驟 1：打開遊戲
1. 在瀏覽器中打開 `index.html`
2. 或使用本地服務器（推薦）：
   ```bash
   # 如果你有 Python
   python -m http.server 8000
   # 然後在瀏覽器訪問 http://localhost:8000
   ```

### 步驟 2：創建角色
1. 點擊「新建角色」按鈕
2. 輸入角色名稱
3. 選擇種族和職業
4. 點擊「擲骰分配」自動分配屬性（或手動輸入）
5. 點擊「保存角色」

### 步驟 3：開始冒險
1. 點擊「開始冒險」
2. 選擇一個故事
3. 使用預設按鈕或輸入自定義動作
4. 享受你的冒險！

## 🤖 AI 設置（可選但推薦）

### 使用 OpenAI（簡單版）

1. **獲取 API 密鑰**
   - 訪問 https://platform.openai.com
   - 註冊/登錄
   - 點擊「Create new secret key」

2. **配置遊戲**
   - 點擊「設置」
   - API 提供商選擇「OpenAI」
   - 貼上你的 API 密鑰
   - 點擊「保存設置」

3. **開始遊戲**
   - 現在 AI 會生成更動態的故事！

### 不配置 API 也可以玩！

遊戲的離線模式提供預設回應，讓你也能享受基本的冒險體驗。

## 💾 保存與加載

- **自動保存**：遊戲每 30 秒自動保存進度
- **角色保存**：角色永久存儲在本地瀏覽器
- **導出備份**：在設置中點擊「匯出遊戲數據」

## 🔧 故障排除

### 問題：頁面無法加載
**解決方案**：
- 檢查 JavaScript 是否啟用
- 嘗試在新標籤頁中打開
- 清除瀏覽器快取

### 問題：API 不工作
**解決方案**：
- 確認 API 密鑰是否正確
- 檢查網絡連接
- 查看瀏覽器控制台（F12）的錯誤信息

### 問題：數據丟失
**解決方案**：
- 清空 LocalStorage 會丟失數據
- 定期匯出數據作為備份
- 使用匯出的 JSON 文件恢復數據

## 📞 快速幫助

## 🌟 高級技巧

### 修改屬性（開發者工具）
```javascript
// 在 F12 控制台運行
const char = characterManager.getCurrentCharacter();
char.stats.str = 20; // 修改力量
characterManager.updateCharacter(char.id, char);
```

