🎯 快速開始（3 步）

1️⃣  打開遊戲
   在瀏覽器中打開: index.html

2️⃣  創建角色
   點擊「新建角色」→ 選擇種族和職業 → 分配屬性 → 保存

3️⃣  開始冒險
   點擊「開始冒險」→ 選擇故事 → 描述你的行動

⚙️  配置 AI（可選，推薦 OpenAI）

1. 訪問 https://platform.openai.com
2. 創建帳戶並生成 API 密鑰
3. 點擊遊戲「設置」
4. 選擇 OpenAI，貼上密鑰
5. 點擊「保存設置」
6. 完成！享受 AI 生成的故事

💡 常見問題

Q: 我沒有 API 密鑰可以玩嗎？
A: 可以！離線模式提供預設故事回應。

Q: 我的數據在哪裡？
A: 存儲在瀏覽器 LocalStorage 中。清除瀏覽器數據會刪除遊戲進度。

Q: 怎樣備份遊戲進度？
A: 設置 → 「匯出遊戲數據」 → 下載 JSON 文件

Q: 支持多人遊戲嗎？
A: 目前版本是單人。未來版本可能支持。

Q: 遊戲費用是多少？
A: 完全免費（如果使用離線模式）。
   使用 OpenAI 需要付費，但成本很低。

Q: 我可以添加自己的故事嗎？
A: 可以！編輯 js/game.js 中的故事配置。
